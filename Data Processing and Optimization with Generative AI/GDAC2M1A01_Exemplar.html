# Add a column indicating if the data is real or generated
df['source'] = 'real'

# Defining the number of samples
total_samples = 100

# Separate features and target
X = df[['heart_rate', 'oxygen_saturation', 'high_blood_pressure', 'low_blood_pressure']]
y = df['health_status']

# Apply SMOTE to balance the dataset
smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X, y)

# Generate additional samples to create 100 rows
while len(X_resampled) < total_samples:
    X_additional, y_additional = smote.fit_resample(X, y)
    X_resampled = pd.concat([pd.DataFrame(X_resampled), pd.DataFrame(X_additional)], ignore_index=True)
    y_resampled = pd.concat([pd.Series(y_resampled), pd.Series(y_additional)], ignore_index=True)
    # Trim the excess rows if necessary
    if len(X_resampled) > total_samples:
        X_resampled = X_resampled[:total_samples]
        y_resampled = y_resampled[:total_samples]

# Create a new balanced DataFrame with 100 rows
balanced_df = pd.DataFrame(X_resampled, columns=['heart_rate', 'oxygen_saturation', 'high_blood_pressure', 'low_blood_pressure'])
balanced_df['health_status'] = y_resampled
balanced_df['source'] = 'generated'

The Kolmogorov-Smirnov (KS) test is a non-parametric test used to compare the distributions of two datasets.

The KS test compares the entire distribution, considering differences in shape, central tendency, and variability. This makes it a better choice when we want to evaluate whether two datasets are drawn from the same underlying distribution, regardless of their form.

In the KS test, the null hypothesis is:

"The two datasets come from the same distribution."

This means that if the null hypothesis is true, there should be no significant differences in the shapes, central tendencies, or variability of the two distributions being compared.

If the test returns a low p-value (typically below 0.05), you reject the null hypothesis, concluding that the two datasets are not drawn from the same distribution.

If the p-value is high, you fail to reject the null hypothesis, meaning there is no strong evidence to suggest that the distributions are different. They could be from the same distribution.

In[]:
from scipy.stats import ks_2samp

# Statistical Analysis: Kolmogorov-Smirnov Test
print("KS Test for Numerical Features")
alpha = 0.05  # Significance level

for col in numerical_columns:
    ks_stat, p_value = ks_2samp(real_data[col], generated_data[col])
    conclusion = (
        "Reject the null hypothesis (distributions are significantly different)"
        if p_value < alpha
        else "Fail to reject the null hypothesis (no significant difference in distributions)"
    )
    print(f"{col}: KS statistic = {ks_stat:.2f}, p-value = {p_value:.2f} - {conclusion}")
